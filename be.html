等效電阻計算

並聯電阻等效電阻公式：1/Req = 1/R1 + 1/R2 + ... + 1/Rn

串聯電阻等效電阻公式：Req = R1 + R2 + ... + Rn

求電路總電阻

輸入各個電阻值來計算總電阻。

電壓升與電壓降

在閉合迴路中，電壓升等於電壓降。使用基爾霍夫電壓定律（Kirchhoff's Voltage Law）進行計算。

分壓原理

分壓原理應用於串聯電阻。公式：Vx = (Rx / Rtotal) * Vtotal

消耗功率

電功率公式：P = V * I 或 P = I2 * R 或 P = V2 / R
